# 字符战争

## 休闲模式

### 创建初始项目

1. 创建空白.net core项目
    - [x] 创建空白项目
    - [x] 创建单元测试项目
2. 创建初始界面
    > 统一语言：
    >
    > 棋盘: Checkerboard
    >
    > 当前候选棋子: CurrentCandidate
    > 
    > 下一个候选棋子: NextCandidate
    
    - [x] 创建空白棋盘
    
        > 空白棋盘按照如下格式显示：
        > ``` bash
        > [  ] [  ] [  ] [  ]
        > [  ] [  ] [  ] [  ]
        > [  ] [  ] [  ] [  ]
        > [  ] [  ] [  ] [  ]
        > ```
    
    - [x] 创建候选数字1和候选数字2
    
        > 候选数字显示在棋盘下方，整体显示格式如下：
        > ```bash
        > [  ] [  ] [  ] [  ]
        > [  ] [  ] [  ] [  ]
        > [  ] [  ] [  ] [  ]
        > [  ] [  ] [  ] [  ]
        > Next: 1; Current: 1
        > ```
